Palette

These are the colors that get mapped to such classes as .button, .hero, .box, .notification, .message, .burst, .badge, .loaders, .progress . They can be changed in variables.scss. Use it like this <div class="box is-primary"></div>.

Icons

Icons are font based and built using icomoon.

Use <i class="ics-name"></i> switching in the desired icon's name.

Currently it's a custom set I put together for casadereyes.net, but I will be updating soon with more (like more standard social icons).

Backgrounds

In variables.scss you can assign background images and patterns to use. SVG, JPG, PNG, GIF all work. With the exception of has-bkd-pattern, they are all centered and set to cover the div.

Use like this: <section> class="section has-flxs-hero-1"></section>
